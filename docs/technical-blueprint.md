# Wayfarer Technical Blueprint

## Local development environment
- Mac M1
- VS Code
- Terminal / bash

## Core stack
- Frontend: Next.js + TypeScript
- Backend: FastAPI + Python 3.11
- Database: PostgreSQL + pgvector
- Cache / jobs: Redis
- AI providers: OpenAI gpt-4o-mini + Ollama
- External APIs: Tripadvisor Content API + Google Places API

## Development rules
- Stepwise implementation
- Full code files only
- No patch-based changes
- Existing files must be shared before modification
- Backend and frontend confirmations required before proceeding
