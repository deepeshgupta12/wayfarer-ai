# Wayfarer Product Scope

Wayfarer is an AI-powered travel discovery, planning, and on-ground intelligence platform that helps users discover destinations, compare them, build personalized itineraries, get contextual recommendations during travel, and improve future recommendations through memory and feedback loops.

This is not a flight-booking or hotel-booking platform in the current scope.

It is a travel intelligence and planning platform powered by:
- Tripadvisor Content API
- Google Places API
- LLM orchestration
- vector search
- persona modeling
- trip memory
- agentic planning and re-planning
