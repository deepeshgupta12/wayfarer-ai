# Wayfarer

Wayfarer is an AI-powered travel discovery, planning, and on-ground intelligence platform.

## Frozen product versions

### V1 — Personalized Destination Intelligence
- Traveller Persona Engine
- Review Intelligence Layer
- Location Embedding Index
- Conversational Destination Guide

### V2 — Personalized Trip Planning
- Master Travel Agent
- Similarity-Based Alternatives
- Destination Comparison Agent

### V3 — Contextual Travel Companion
- Hidden Gem Detector
- Context-Aware On-Ground Finder
- Proactive Notification Agent
- Photo Intelligence

### V4 — Compounding Intelligence Platform
- Trip Memory & Feedback Loop
- Evolving Traveller Model
- B2B2C API + white-label
- Review Authenticity Moat

## Monorepo structure

- `apps/api` — backend application
- `apps/web` — frontend application
- `packages/prompts` — prompt assets and reusable prompt definitions
- `packages/schemas` — shared contracts and schemas
- `packages/ui-spec` — UX, IA, and UI system documentation
- `infra/docker` — local infrastructure setup
- `docs` — product and engineering documentation
- `scripts` — helper scripts

## Working rules

- Stepwise implementation only
- Full code files only
- No code patches
- No hallucinated edits
- Existing files must be shared in full before modification
- Backend and frontend must be confirmed before moving to the next step
